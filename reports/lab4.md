### Изучение особенностей дистрибутива

1. Были выяснены следующие детали про дистрибутив по варианту B:

  - организация занимающаяся разработкой и поддержкой дистрибутива -- Manjaro GmbH & Co;
  - базовый дистрибутив -- независимый дистрибутив Linux, построенный на базе Arch Linux;
  - как часто выпускаются релизы -- Manjaro Linux использует модель rolling release (непрерывный цикл обновлений), Пакеты обновлений выпускаются примерно каждые 2 недели, последним крупным релизом является Manjaro Linux 26.0 «Anh-Linh», выпущенный 4 января 2026 года;
  - ресурсы изучения деталей работы с дистрибутивом -- множество ресурсов, начиная от официальной документации и заканчивая активным сообществом. Например, [местная вики](https://wiki.manjaro.org/index.php/Main_Page).

2. Manjaro использует пакетный менеджер Pamac. По умолчанию система использует pamac в качестве менеджера управления программами, которая является графическим вариантом консольной утилиты pacman. [Источник](https://losst.pro/ustanovka-programm-v-manjaro#ustanovka-programm-v-manjaro-linux)

Чтобы запустить его, необходимо открыть меню.

Pamac имеет схожий с подобными программами интерфейс, в котором есть возможность сортировать все программы по категориям, группам и т.д.

Установленные программы имеют окрашенное зеленым цветом поле Состояние. Чтобы установить программу Manjaro, надо выбрать необходимый фильтр ПО (например Категории). 

Выбрать одну или несколько программ для установки, поставив галочку в поле `Состояние -> Применить`.

При необходимости установить дополнительные зависимости ПО, pamac выдаст соответствующее окно с полным их перечнем.

При первой попытке установки, за время работы pamac будет запрошен пароль пользователя. Необходимо ввести его и нажать Аутентифицировать.

Начнётся инсталляция. Для просмотра подробностей установки необходимо нажать кнопку Подробно.

По окончании процесса появится сообщение об успешном завершении транзакции с соответствующим уведомлением.

3. Было выяснено, как происходит процесс установки дистрибутива. Подробная инструкция была найдена на [Яндекс дзене](https://dzen.ru/a/Ztz_4G-dXQGiDspP). Для установки операционной системы Manjaro KDE необходимо подготовить установочный образ системы в формате iso. Его можно скачать с [официального сайта](https://manjaro.org/products/download/x86). Также, предварительно, необходимо установить [VirtualBox](https://www.virtualbox.org/), это бесплатное программное обеспечение для виртуализации, позволяющее запускать несколько гостевых операционных систем внутри основной (хостовой) системы одновременно. Далее следовать инструкции с Дзена. 

4. Системные требования для [Manjaro](https://wiki.manjaro.org/index.php/About_Manjaro/en#Recommended_System_Requirements): 4 ГБ ОЗУ, 30 ГБ на диске, двухъядерный процессор, HD-совместимая графика. Эти же условия относятся и к оконному менеджеру.

```

Папка ВМ: C:\Users\Тот Самый\VirtualBox VMs

Редакция ОС: Linux
Дистрибутив ОС: ArchLinux
Версия ОС: Arch Linux (64-bit)
```
В следующем окне с выбором виртуального оборудования было выставлено следующее, в соответствии с рекомндуемыми (EFi -> современный режим загрузки):
```
Основная память: 8 гб
Число ЦПУ: 4
Размер диска: 30 гб
Галочка "использовать EFI"
```
